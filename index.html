// sobre mí

Desarrollo software que funciona,
desde la idea hasta producción.

Soy un Desarrollador de Software Full Stack con 8 años de experiencia construyendo aplicaciones Web, Desktop y Móviles. A lo largo de mi trayectoria he asumido roles de Líder Técnico, guiando equipos de desarrollo y apoyando en la definición de arquitecturas de sistemas robustos y escalables.

Tengo sólida experiencia en el despliegue de aplicaciones en servidores, automatización con GitHub Actions, orquestación de contenedores con Docker y manejo de múltiples herramientas del ecosistema moderno de desarrollo.

Actualmente desarrollo apps móviles de manera independiente bajo mi marca GB Solutions Apps, buscando resolver problemas reales con software de calidad y buen diseño.
